# Formulaire connexion Make

Un simple formulaire HTML conçu pour envoyer des données à un webhook Make.com. Déployé sur Vercel.

## Comment l'utiliser

1. Remplacez l'URL du webhook dans le fichier `index.html` par la vôtre.
2. Déployé via Vercel ou GitHub Pages.
